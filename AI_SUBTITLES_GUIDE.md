# AI 字幕生成功能使用指南

## ✨ 功能概述

系统现在支持使用 **OpenAI Whisper API** 和 **百度翻译 API** 自动为视频生成：
1. **英文字幕** - 语音识别（ASR）
2. **中文字幕** - 自动翻译
3. **双语字幕** - 英中对照

---

## 🚀 使用步骤

### 1. 获取 API Key

#### OpenAI API Key（语音识别）
1. 访问 [OpenAI Platform](https://platform.openai.com/api-keys)
2. 登录或注册账号
3. 创建新的 API Key
4. 复制 API Key（格式：`sk-...`）

#### 百度翻译 API Key（翻译）
1. 访问 [百度翻译开放平台](https://fanyi-api.baidu.com/)
2. 注册/登录百度账号
3. 进入「管理控制台」→「开发者信息」
4. 查看你的 **APP ID** 和 **密钥**
5. 复制这两个信息（格式：`APP_ID,SECRET_KEY`）

### 2. 上传视频

1. 访问管理后台：`http://localhost:3005/admin`
2. 点击"上传视频"按钮
3. 填写视频信息并上传视频文件
4. **无需上传字幕文件**

### 3. 生成字幕

1. 在视频列表中找到没有字幕的视频
2. 点击视频卡片上的 **"AI 生成字幕"** 按钮
3. 输入 API Key，格式：`OpenAI Key, 百度 APP_ID,SECRET_KEY`
   - 例如：`sk-abc123...,12345678,abcd1234efgh5678`
4. 点击"开始生成"
5. 等待处理完成（通常 1 分钟视频需要 10 秒）

### 4. 查看结果

生成完成后：
- 视频会显示有 2 个字幕（英文 + 中文）
- 可以在视频播放页面查看双语字幕
- 字幕自动同步并可以点击跳转

## ⚙️ 技术细节

### 工作流程

```
视频文件
  ↓
提取音频 (ffmpeg)
  ↓
OpenAI Whisper API 语音识别
  ↓
生成英文字幕 (SRT)
  ↓
百度翻译 API 翻译（逐句）
  ↓
生成中文字幕 (SRT)
  ↓
合并双语字幕
  ↓
保存到数据库
```

### 支持的视频格式

- MP4, WebM, OGG, MOV, AVI, MKV

### 字幕格式

- **SRT 格式** - 标准 SubRip 格式
- **时间码精度** - 毫秒级
- **编码** - UTF-8

### API 限制

- **OpenAI Whisper**：无速率限制，按使用量计费
- **百度翻译标准版**：
  - QPS=1（每秒 1 次请求）
  - 每月 5 万字符免费
- **文件大小**：建议 < 100MB

## 🔧 故障排除

### 问题 1：提取音频失败

**原因**：系统未安装 ffmpeg

**解决方案**：
```bash
# Windows: 下载并添加到 PATH
# https://ffmpeg.org/download.html
```

### 问题 2：Whisper API 调用失败

**可能原因**：
1. API Key 无效
2. 账户余额不足
3. 网络连接问题

**解决方案**：
- 检查 OpenAI API Key 是否正确（格式：`sk-...`）
- 确认 OpenAI 账户余额
- 检查网络连接（可能需要代理）

### 问题 3：百度翻译 API 调用失败

**可能原因**：
1. APP ID 或密钥错误
2. 签名验证失败
3. 超出免费额度或配额

**解决方案**：
- 检查 APP ID 和密钥是否正确
- 确认格式为：`APP_ID,SECRET_KEY`
- 确认百度账户配额

### 问题 4：翻译速度慢

**原因**：百度翻译 QPS=1 限制

**解决方案**：
- 这是正常现象，系统会自动处理速率限制
- 预计翻译时间 ≈ 字幕片段数量 × 1 秒

### 问题 5：生成时间过长

**原因**：视频太长或服务器繁忙

**预计时间**：
- 1 分钟视频 ≈ 10 秒（识别） + 30 秒（翻译）
- 5 分钟视频 ≈ 30 秒（识别） + 2 分钟（翻译）
- 10 分钟视频 ≈ 1 分钟（识别） + 4 分钟（翻译）
