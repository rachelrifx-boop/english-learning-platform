# Google Cloud Speech-to-Text 配置指南

本指南将帮助您配置 Google Cloud Speech-to-Text API，用于英语学习平台的 AI 字幕生成功能。

## 为什么选择 Google Cloud Speech-to-Text？

### 费用对比（200个视频 × 10分钟 = 2000分钟）

| 服务 | 价格 | 免费额度 | 总费用 |
|------|------|---------|--------|
| **OpenAI Whisper** | $0.006/分钟 | 无 | $12 (约85元) |
| **Google Cloud** | $0.004/分钟 | 每月60分钟 | **$7.76** (约55元) |
| **新用户优惠** | - | $300赠金 | **完全免费** ✨ |

### 推荐方案
对于您的项目（200个10分钟视频），强烈推荐使用 **Google Cloud Speech-to-Text**：
- 新用户 $300 赠金足够处理 **75,000分钟**（1,250小时）
- 剩余部分只需 $7.76
- 配合百度翻译API（免费），总成本接近 $0

## 步骤 1：创建 Google Cloud 项目

1. 访问 [Google Cloud Console](https://console.cloud.google.com/)
2. 点击顶部的项目选择器
3. 点击 **"新建项目"**
4. 输入项目名称（例如：`english-learning-platform`）
5. 点击 **"创建"**

---

## 步骤 2：启用 Speech-to-Text API

1. 在 Google Cloud Console 中，确保您选择了刚创建的项目
2. 打开导航菜单，进入 **"API 和服务"** → **"库"**
3. 搜索 **"Cloud Speech-to-Text API"**
4. 点击进入，然后点击 **"启用"**

---

## 步骤 3：创建 API Key

### 方法 A：使用 API Key（推荐，快速开始）

1. 进入 **"API 和服务"** → **"凭据"**
2. 点击 **"创建凭据"** → **"API 密钥"**
3. 复制生成的 API Key（格式：`AIza...`）
4. **重要**：为了安全，建议限制 API Key
   - 点击 **"编辑 API 密钥"**
   - 在 **"应用限制"** 中，选择您网站的域名
   - 在 **"API 限制"** 中，只选择 **"Cloud Speech-to-Text API"**

### 方法 B：使用服务账号密钥（生产环境推荐）

1. 进入 **"IAM 和管理员"** → **"服务账号"**
2. 点击 **"创建服务账号"**
3. 输入服务账号名称（例如：`speech-to-text-sa`）
4. 点击 **"创建并继续"**
5. 角色：选择 **"Cloud Speech-to-Text API 用户"**
6. 点击 **"完成"**
7. 创建完成后，点击服务账号邮箱
8. 切换到 **"密钥"** 标签
9. 点击 **"添加密钥"** → **"创建新密钥"**
10. 选择 **"JSON"** 格式，点击 **"创建"**
11. 下载的 JSON 文件包含您的私钥，**请妥善保管**

**注意**：如果使用服务账号 JSON，您需要提取 `project_id` 和使用私钥认证。对于简单使用，推荐使用方法 A。

---

## 步骤 4：设置计费（首次使用）

即使有免费额度，Google Cloud 仍需要绑定信用卡：

1. 进入 **"计费"** 页面
2. 点击 **"关联结算账号"**
3. 按提示绑定信用卡
4. **不用担心**：
   - 新用户有 $300 赠金
   - 每月60分钟免费额度
   - 只有在超出额度后才会实际扣费
   - 可以设置预算告警，防止意外扣费

---

## 步骤 5：获取 API Key

完成上述步骤后，您将获得一个 API Key，格式类似：
```
AIzaSyCxxxxxxx...
```

---

## 步骤 6：在平台中使用

1. 打开您的英语学习平台（http://localhost:3002）
2. 进入 **"视频管理"**
3. 点击视频卡片上的 **"AI 生成字幕"** 按钮
4. 在弹出的对话框中：
   - **选择语音识别服务**：选择 **"Google Cloud Speech-to-Text"**
   - **输入 API Key**：
     ```
     Google Cloud Key, 百度 APP_ID,SECRET_KEY
     ```
     例如：
     ```
     AIzaSyCxxxxxxx, 12345678, abcd1234
     ```
5. 点击 **"开始生成"**

### 格式说明
- **Google Cloud Key**: 您刚创建的 API Key
- **百度 APP_ID**: 百度翻译 API 的 APP ID
- **百度 SECRET_KEY**: 百度翻译 API 的密钥

## 常见问题

### Q: 会超出预算吗？
A: 不会。您的项目只需 $7.76（约55元），加上新用户 $300 赠金，完全不用担心。

### Q: 必须绑定信用卡吗？
A: 是的，但实际不会扣费（除非超出赠金和免费额度）。建议设置预算告警。

### Q: 如何设置预算告警？
1. 进入 **"计费"** → **"预算和提醒"**
2. 点击 **"创建预算"**
3. 设置金额（例如 $10）
4. 添加提醒规则（达到50%、90%、100%时发送邮件通知）

### Q: API Key 安全吗？
A: 在前端输入的 API Key 仅用于单次调用，不会被保存到数据库。但建议您：
- 设置 API Key 的应用限制（只允许您的域名）
- 定期轮换 API Key
- 不要在公共场合分享

### Q: 如果翻译失败怎么办？
A: 检查以下几点：
1. API Key 是否正确
2. 是否已启用 Speech-to-Text API
3. 是否已设置计费账号
4. 是否超出配额限制

### Q: 处理速度如何？
A: 大约 **1分钟视频需要10-15秒**处理时间。10分钟视频大约需要1.5-2分钟。
